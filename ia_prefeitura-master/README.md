Prefeitura IA
=============

Estrutura do projeto:
- data/ : dados em CSV (dados.csv)
- src/ : código fonte (data_loader.py, analise.py, chat.py, relatorio.py, main.py)

Como rodar (no PyCharm ou terminal):
1. Instale dependências: pip install -r requirements.txt
2. No terminal, execute: python -m src.main
3. Pergunte ao chat: exemplos -> 'gasto saúde 2020', 'média educação', 'resumo geral'
